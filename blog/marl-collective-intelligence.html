多智能体强化学习：群体智能的计算框架Multi-Agent Reinforcement Learning: A Framework for Collective Intelligence

2026-06-18

鸟群如何在没有领导者的情况下协调行动？交通模式如何自组织？成千上万个 AI 智能体如何学会合作、竞争或协调？这些问题都指向同一个根本问题：理解从去中心化交互中涌现的集体行为。多智能体强化学习（MARL）正是研究这一问题的计算框架。

从单智能体到多智能体 RL

在标准（单智能体）强化学习中，智能体学习一个策略 π(a|s)，将状态映射到动作以最大化累积奖励。环境通常是平稳的——转移动态 P(s'|s,a) 与奖励函数 R(s,a) 不随时间变化。

MARL 打破了这一假设。当多个智能体同时学习时，每个智能体的策略更新都会改变其他智能体所感知的环境。环境变得非平稳，马尔可夫性质不再成立。这既是挑战，也是——对我们这些对复杂系统感兴趣的人来说——关键所在。

形式上，马尔可夫博弈（随机博弈）将 MDP 扩展到 N 个智能体：

(S, {A_i}_{i=1}^N, P, {R_i}_{i=1}^N, γ)

其中：
  S     — 共享状态空间
  A_i   — 智能体 i 的动作空间
  P     — 转移：S × A_1 × ... × A_N → Δ(S)
  R_i   — 智能体 i 的奖励函数
  γ — 折扣因子

每个智能体旨在最大化自身的期望回报。根据奖励结构的不同，智能体可能面临完全合作（共享 R）、完全竞争（零和 R）或混合（一般和）场景。

MARL 的核心方法

1. 独立学习器（IL）

最简单的方法：每个智能体独立运行自己的 RL 算法，将其他智能体视为环境的一部分。典型算法包括独立 Q-Learning（IQL）或独立 PPO（IPPO）。

# 概念示意：独立 Q-Learning
for 每个智能体 i 并行执行：
    观测 s_i，执行动作 a_i ~ ε-贪心(Q_i)
    收到 r_i，观测 s_i'
    Q_i(s_i, a_i) ← Q_i(s_i, a_i) + α[r_i + γ max Q_i(s_i', a') - Q_i(s_i, a_i)]

优点：简单、可扩展、无通信开销。
缺点：非平稳性破坏了收敛保证。但在实践中效果出奇地好。

2. 集中训练、分布执行（CTDE）

CTDE 是现代 MARL 的主流范式。在训练阶段，智能体可以访问全局信息（其他智能体的观测、动作，甚至完整状态）。在执行阶段，每个智能体仅使用自身的局部观测采取行动。

代表性算法：

  • MADDPG（Lowe et al., 2017）：每个智能体拥有一个以所有智能体动作为条件的集中式评论家 Q(s, a_1, ..., a_N)，以及仅使用局部观测的分布式演员 π(a_i | o_i)。评论家在执行时丢弃。
  • QMIX（Rashid et al., 2018）：在合作场景中，将联合 Q 函数 Q_tot 分解为各智能体 Q_i 值的单调组合。混合网络强制 ∂Q_tot / ∂Q_i ≥ 0，确保个体与集体目标的一致性。
  • MAPPO（Yu et al., 2022）：将 PPO 扩展至多智能体场景，使用集中式价值函数。实现简单，实证表现强劲。

3. 通信与涌现协调

最令人兴奋的方向之一：智能体可以学会通信。在 CommNet 或 RIAL 等框架中，智能体向邻居发送连续或离散消息。经过训练后，通信协议涌现出来——没有人设计了这门"语言"，它之所以演化，是因为它帮助智能体协调行动。

这与理解人类通信系统如何涌现和稳定直接相关。与演化博弈论中的信号博弈有惊人的相似之处。

为什么 MARL 对复杂系统研究很重要

传统的智能体建模（ABM）使用手工设计的的行为规则（如 Boids 模型："对齐、凝聚、分离"）。MARL 将这些规则替换为学习得到的行为。我们不再指定智能体应该做什么，而是指定它们应该达成什么（奖励），让它们自行探索如何做到。

这打开了若干前沿方向：

  • 涌现：当我们改变种群规模、网络拓扑或奖励结构时，集体行为中是否能观察到相变？
  • 适应：MARL 种群如何应对环境冲击或对抗性扰动？
  • 规范涌现：在什么条件下，合作规范（公平、轮流、惩罚）能从纯粹自利的学习中涌现？
  • 信息扩散：部分可观测性和局部通信如何塑造学习者网络中的信念传播？

一个简单实验：网格世界中的协调

考虑一个最小化实验场景：N 个智能体在网格上，每个必须到达目标。目标是共享的，但智能体不能占据同一格子。奖励：到达目标 +10，每步 −1，碰撞 −5。

# 网格世界 MARL 实验伪代码
class GridWorldMARL:
    def step(self, actions):
        # 执行动作，解决碰撞
        # 智能体具有部分可观测性（看到 3×3 邻域）
        rewards = []
        for i in range(self.n_agents):
            if 到达目标(智能体_i):   rewards.append(10)
            elif 碰撞(智能体_i):    rewards.append(-5)
            else:                  rewards.append(-1)
        return observations, rewards

会涌现什么？种群规模较小时，智能体学会轮流。规模较大时，拥堵模式和排队行为自然出现——没有人编写这些行为，它们来自奖励结构和多智能体动态。

How does a flock of birds coordinate without a leader? How do traffic patterns self-organize? How might thousands of AI agents learn to cooperate, compete, or coordinate? These questions all point to the same underlying problem: understanding collective behavior emerging from decentralized interactions. Multi-Agent Reinforcement Learning (MARL) provides a computational framework to study exactly this.

From Single-Agent to Multi-Agent RL

In standard (single-agent) reinforcement learning, an agent learns a policy π(a|s) that maps states to actions to maximize cumulative reward. The environment is typically stationary — the transition dynamics P(s'|s,a) and reward function R(s,a) don't change over time.

MARL breaks this assumption. When multiple agents learn simultaneously, each agent's policy update changes the environment from the perspective of every other agent. The environment becomes non-stationary, and the Markov property no longer holds. This is both a challenge and — for those of us interested in complex systems — the whole point.

Formally, a Markov game (stochastic game) extends the MDP to N agents:

(S, {A_i}_{i=1}^N, P, {R_i}_{i=1}^N, γ)

where:
  S     — shared state space
  A_i   — action space of agent i
  P     — transition: S × A_1 × ... × A_N → Δ(S)
  R_i   — reward function for agent i
  γ — discount factor

Each agent aims to maximize its own expected return. Depending on how rewards are structured, agents may face fully cooperative (shared R), fully competitive (zero-sum R), or mixed (general-sum) settings.

Key Approaches in MARL

1. Independent Learners (IL)

The simplest approach: each agent runs its own RL algorithm, treating other agents as part of the environment. Think Independent Q-Learning (IQL) or Independent PPO (IPPO).

# Conceptual sketch: Independent Q-Learning
for each agent i in parallel:
    observe s_i, take action a_i ~ ε-greedy(Q_i)
    receive r_i, observe s_i'
    Q_i(s_i, a_i) ← Q_i(s_i, a_i) + α[r_i + γ max Q_i(s_i', a') - Q_i(s_i, a_i)]

Pros: Simple, scalable, no communication overhead.
Cons: Non-stationarity from the learner's perspective breaks convergence guarantees. Works surprisingly well in practice though.

2. Centralized Training with Decentralized Execution (CTDE)

CTDE is the dominant paradigm in modern MARL. During training, agents can access global information (other agents' observations, actions, even the full state). During execution, each agent acts using only its own local observation.

Representative algorithms:

  • MADDPG (Lowe et al., 2017): Each agent has a centralized critic Q(s, a_1, ..., a_N) that conditions on all agents' actions, but a decentralized actor π(a_i | o_i) that uses only local observation. The critic is discarded at execution time.
  • QMIX (Rashid et al., 2018): For cooperative settings, factorizes the joint Q-function Q_tot as a monotonic combination of per-agent Q_i values. A mixing network enforces ∂Q_tot / ∂Q_i ≥ 0, ensuring consistency between individual and collective goals.
  • MAPPO (Yu et al., 2022): Extends PPO to multi-agent settings with a centralized value function. Simple to implement, strong empirical performance.

3. Communication and Emergent Coordination

One of the most exciting directions: agents can learn to communicate. In frameworks like CommNet or RIAL, agents send continuous or discrete messages to neighbors. Over training, communication protocols emerge — no one designed the language, it evolved because it helped agents coordinate.

This is directly relevant to understanding how human communication systems might emerge and stabilize. The parallels to signaling games in evolutionary game theory are striking.

Why MARL Matters for Complex Systems Research

Traditional agent-based models (ABMs) use hand-crafted behavioral rules (e.g., Boids: "align, cohere, separate"). MARL replaces these with learned behaviors. Instead of specifying what agents should do, we specify what they should achieve (the reward), and let them figure out how.

This opens several frontiers:

  • Emergence: Can we observe phase transitions in collective behavior as we vary population size, network topology, or reward structure?
  • Adaptation: How do MARL populations respond to environmental shocks or adversarial perturbations?
  • Norm emergence: Under what conditions do cooperative norms (fairness, turn-taking, punishment) emerge from purely self-interested learning?
  • Information diffusion: How does partial observability and local communication shape belief propagation through a network of learners?

A Simple Experiment: Coordination in a Grid World

To make this concrete, consider a minimal setup: N agents on a grid, each must reach a goal. Goals are shared but agents cannot occupy the same cell. Reward: +10 for reaching a goal, −1 per step, −5 for collision.

# Pseudocode for grid-world MARL experiment
class GridWorldMARL:
    def step(self, actions):
        # Apply actions, resolve collisions
        # Agents share partial observability (see 3×3 neighborhood)
        rewards = []
        for i in range(self.n_agents):
            if at_goal(agent_i):   rewards.append(10)
            elif collision(agent_i): rewards.append(-5)

What emerges? With small populations, agents learn to take turns. With larger populations, congestion patterns and queueing behavior emerge — no one programmed these behaviors, they arose from the reward structure and multi-agent dynamics.

Open Questions I'm Thinking About

  1. Scalability: How do learned collective behaviors change as we scale from dozens to thousands of agents? Most MARL benchmarks use N ≤ 10.
  2. Heterogeneity: Real populations are diverse in capabilities, goals, and information. How does heterogeneity affect emergent coordination?
  3. From ABM to MARL: Can we re-implement classic ABM models (Schelling segregation, Sugarscape, SIR) in MARL and observe richer dynamics?
  4. Bridging levels: How do we formally characterize the micro-macro link — from individual Q-functions to population-level equilibria?
