如何撰写博客文章How to Write a Blog Post

第 1 步：用 Markdown 撰写文章，顶部使用以下格式：

---
title: "文章标题"
date: 2026-06-15
---

正文内容。使用标准 Markdown：

## 章节标题

- 列表项
- `行内代码`

```python
# 代码块
print("hello")
```

数学公式：$f(x) = x^2$

第 2 步：转换为 HTML。在 blog/ 目录中运行：

conda activate writing-tools
python -c "
import markdown, sys
text = open('your-post.md').read()
html = markdown.markdown(text, extensions=['fenced_code', 'codehilite', 'tables'])
title = text.split('\n')[1].replace('title: ','').strip('\"')
date  = text.split('\n')[2].replace('date: ','').strip()
template = open('_template.html').read()
template = template.replace('{{TITLE}}', title)
template = template.replace('{{DATE}}', date)
template = template.replace('{{CONTENT}}', html)
open('your-post.html','w').write(template)
print('Generated your-post.html')
"

第 3 步：在本页面（blog/index.html）中添加新文章链接，提交并推送。
